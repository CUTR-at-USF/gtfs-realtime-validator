Welcome to GTFS-realtime Validator

What is the GTFS-Realtime Validator?

This tool takes a GTFS and GTFS-rt feeds as inputs, and examines the GTFS-rt feed to determine if it properly meets GTFS-rt best practices, both those explicitly listed in the spec and those based on the needs of consuming applications.

Getting Started

  1. Input a GTFS feed link (Having at least one feed is mandatory)
  2. Press " Add Feed" if more feed needs to be monitored
  3. The " Remove Feed" can be used for removing any fields added accidentally
  4. Add the associated GTFS feed link on the "GTFS feed" field.
  5. Press the start button to start the monitoring process.
  6. Reset button can be used to reset all fields.
